# Ecoleta---NextLevelWeek

Primeiro projeto realizado, onde pude colocar em prática os ensinamentos de:
HTML + Nunjks
CSS
JS
SQlite
e o meu primeiro contato com o NodeJS.

Nesse projeto, em comemoração a semana da coleta, foi desenvolvido um sistema onde era possível cadastrar
pontos de coleta e que tipo de coleta esses pontos realizavam, podendo selecionar o estado e a cidade.
Após esse cadastro, era possível pesquisar por cidade os pontos de coleta cadastrados.
